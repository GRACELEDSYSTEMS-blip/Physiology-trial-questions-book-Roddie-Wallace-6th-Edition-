The Process

How it works

1
Choose your chapters
Select one or more of the 12 physiological systems — from Body Fluids to Interpretative Questions. Mix chapters freely for cross-system practice.
2
Set up your session
Choose MCQs, EMQs, or both. Pick Instant Mode for immediate feedback or Submit Mode to simulate exam conditions. Set a custom question count or go through everything.
3

Each MCQ has five True/False branches. Each EMQ presents clinical cases matched to a list of options. Every answer comes with a detailed explanation from the authors.

See your adjusted score, grade band, and chapter-by-chapter breakdown. Review every answer with full explanations before retrying or starting a new session.
Question Formats

Two types of questions

☑
Multiple Choice (MCQ)
A stem statement with five branches (A–E). Each branch is answered True or False independently. All five decisions must be made — answers are not mutually exclusive.
691 questions
🩺
Extended Matching (EMQ)
A list of options followed by clinical case scenarios (A–E). Select the single best option for each case. An option may be used more than once — or not at all.
57 questions
